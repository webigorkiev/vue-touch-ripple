* Now the name of the effect can be passed to the arg of the event
  * v-touch-ripple:**tap**
  * v-touch-ripple:**swipe**
* Upgrade docs
* You can set animation duration as modifier, like: v-touch-ripple:swipe.once.**800** (in ms)
